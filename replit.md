# Cookie Recipe Calculator

## Overview
A simple Python command-line program that calculates the amount of ingredients needed to make a specific number of cookies. The program uses a base recipe for 48 cookies and scales the ingredients proportionally.

## Base Recipe (48 cookies)
- 1.5 cups of sugar
- 1 cup of butter
- 2.75 cups of flour

## Project Structure
- `cookie_calculator.py` - Main program file containing the calculator logic

## How to Run
Run the program with: `python cookie_calculator.py`
